# Circle Agency | MVP de Experiencia Digital

## Descripción General

Circle Agency es una landing page de alto rendimiento diseñada para una agencia de diseño web moderna. Este proyecto fue desarrollado como un Producto Mínimo Viable (MVP), priorizando una interfaz de usuario limpia, una experiencia de usuario fluida y una base técnica sólida.

El objetivo principal fue transformar un sistema de diseño en una aplicación web interactiva y totalmente responsiva utilizando tecnologías nativas y lógica de programación moderna.

## Especificaciones Técnicas

### Tecnologías y Lenguajes

- **HTML5:** Estructura semántica para SEO y accesibilidad.
- **CSS3:** Propiedades personalizadas (variables), Flexbox, CSS Grid y diseño responsivo fluido.
- **JavaScript (ES6+):** Manipulación dinámica del DOM y lógica de validación.

### Integraciones y APIs

- **Consumo de API:** Integración de servicios externos para la carga dinámica de datos y gestión de contenido.

### Entorno de Desarrollo y Herramientas

- **Control de Versiones:** GitHub (Git para la gestión de ramas y commits).
- **Despliegue:** Netlify (CI/CD para integración continua).
- **Editor de Código:** Visual Studio Code.
- **Inteligencia Artificial:** Copilot integrado (modelo Claude) y Gemini para la optimización de flujos de trabajo y resolución de problemas técnicos.

## Características Clave (Perspectiva Técnica)

- **Estrategia Mobile-First:** Desarrollo con enfoque móvil mediante el uso de `clamp()` y unidades relativas para un escalado fluido entre resoluciones.
- **Motor de Formularios:** Validación en tiempo real para el formulario de contacto, asegurando la integridad de los datos antes del envío.
- **Componentes Dinámicos:** \* Navegación interactiva adaptada a dispositivos táctiles.
  - Gestión de capas y profundidad mediante posicionamiento avanzado de elementos visuales.
- **Optimización de Rendimiento:** Estructura modular de archivos para facilitar el mantenimiento y la velocidad de carga.
