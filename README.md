# Proyecto 1 - Circle Agency

## Descripción del proyecto

Circle Agency es una landing page moderna y profesional para una agencia de diseño web. El sitio ofrece una experiencia de usuario fluida, intuitiva y completamente adaptada a cualquier dispositivo.

Diseñado para mostrar nuestro portafolio de proyectos, servicios especializados y facilitar el contacto directo con nuestro equipo.

## Características

- Diseño moderno y atractivo con interfaz intuitiva
- Completamente responsive: se adapta perfectamente a móviles, tablets y escritorios
- Portafolio dinámico con galería de proyectos destacados
- Información detallada de cada proyecto realizado
- Formulario de contacto funcional y seguro
- Navegación fluida entre secciones
- Optimizado para rendimiento y velocidad
- Interfaz accesible para todos los usuarios

## Páginas incluidas

**Inicio**
Presentación de la agencia con propuesta de valor, servicios principales y proyectos destacados.

**Proyectos**
Galería completa de nuestros trabajos con descripción detallada, imágenes y resultados de cada proyecto.

**Contacto**
Formulario para que potenciales clientes se comuniquen directamente con nosotros.

## Configuración

### Requisitos

- Conexión a Internet
